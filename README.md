# Synent Task 9 - Full Stack Task Manager (Advanced Level)

## Developer
Lincoln Adura

## Description
An advanced-level Full Stack Task Manager web application built 
with Python Flask. Features complete user authentication, task 
management with priorities and statuses, filtering system and 
a clean responsive UI. Built using Flask, SQLAlchemy, 
Flask-Login and SQLite.

## Features
- 🔐 User Registration and Login
- 🔒 Secure password hashing with Werkzeug
- 📊 Dashboard with task statistics
- ➕ Add tasks with title, description, priority, status and due date
- ✏️ Update any task
- 🗑️ Delete tasks with confirmation
- 🔍 Filter tasks by status and priority
- 💾 Persistent SQLite database storage
- 🎨 Clean and responsive UI
- 🔒 Session handling and protected routes
- ✅ Full input validation and error handling

## Tech Stack
- **Backend:** Python, Flask
- **Database:** SQLite with SQLAlchemy ORM
- **Authentication:** Flask-Login, Werkzeug
- **Frontend:** HTML, CSS (Jinja2 Templates)

## How to Run
Install required libraries:
```bash
pip install flask flask-sqlalchemy flask-login werkzeug
```
Then run:
```bash
python app.py
```
Open your browser and go to:
[Synent Task Manager-LincolnAdura](http://127.0.0.1:5000/)
